## Overview
This project is a **Phishing Simulator** created with **Flask**. It demonstrates how phishing attacks work and teaches users how to protect their online accounts. Users can see a fake login page, submit credentials, and observe how data could be captured in a real phishing scenario — safely for educational purposes.
---
## Features
- Realistic login page with **HTML/CSS** frontend.
- **Flask backend** to handle input securely (simulated capture).
- Educational simulation of phishing attacks.
- Results display for demonstration without harming anyone.
---
1. The login.html file is the login interface.

2. The result.html file displays the password, username, number of victims, and time.

3. The url-chacker.html file is the URL input interface and checks its security.
